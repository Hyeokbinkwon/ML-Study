CNN의 개인적 이해

1. 선형 시스템의 Convolution 연산

◎ 시불변 선형 시스템에 입력이 주어졌을 때 출력을 구하기 위한 연산
   (출력) = (입력) 과 (시스템이 지닌 함수) 의 Convolution 연산

◎ 입력 1 → 시스템 → 출력 1
◎ 입력 2 → 시스템 → 출력 2
◎ (상수) x 입력 1 → 시스템 → (상수) x 출력 1
◎ 입력 1 + 입력 2 → 시스템 → 출력 1 + 출력 2 

2. 머신러닝의 모델

◎ 입력과 가중치의 곱으로 표현 = 가중치의 선형 함수 
◎ 선형 시스템의 특성을 지님
◎ 입력 하나에 가중치를 곱해서 출력 값 하나가 표현되며, 
    여러 개 입력에 따른 출력 값들을 더해서 하나의 출력을 만들 수 있음
→ 더해진 출력은 각각의 입력 / 가중치 변화에 따라 영향을 줄 수 있음

◎ 여러 입력에 각 가중치를 곱한 후 더하여 하나의 출력을 만드는 시스템을 적용
→ Convolutional Neural Networks

3. CNN : Convolutional Neural Networks

◎ Filter : 여러 입력에 각 가중치를 곱한 후 더하여 하나의 출력을 만드는 시스템
◎ 입력 데이터 셋에 Filter를 겹겹이 적용하여 출력 값을 뽑아내면서 Data 수를 줄임
→ 2번처럼 가중치를 적용하는 모델은 선형이므로 각 입력이 출력에 비례하여 영향
◎ Pooling : Filter를 적용하는데 가중치를 곱하는 것이 아닌, 필터 내 값 중 하나를 추출
   * Max 값을 추출 or Average 값을 추출 등등
→ Overfitting을 방지하기 위해 Convolution 중간중간에 적용
